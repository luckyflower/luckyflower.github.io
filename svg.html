两个属性

stroke-dasharray

stroke-dasharray:在SVG中表示描边是虚线， 可选值为：none, <dasharray>, inherit. 其中，none表示不是虚线；<dasharray>为一个逗号或空格分隔的数值列表,两个值，第一个是虚线的宽度，第二个是虚线之间的间距。表示各个虚线端的长度。可以是固定的长度值，也可以是百分比值；inherit表继承。

stroke-dashoffset

stroke-dashoffset:表示虚线的起始偏移。可选值为：<percentage>, <length>,inherit. 百分比值，长度值，继承。

示例：

<line stroke-dasharray="5, 5"  x1="10" y1="10" x2="190" y2="10" />
<line stroke-dasharray="5, 10"  x1="10" y1="30" x2="190" y2="30" />
<line stroke-dasharray="15, 10, 5" x1="10" y1="130" x2="190" y2="130" />
<line stroke-dasharray="15, 10, 5, 10"  x1="10" y1="150" x2="190" y2="150" />

路径的长度

如果您想知道路径，或线条的准确长度，可以借助JavaScript获取，如下面的代码：

var path = document.querySelector('path');
var length = path.getTotalLength();

制作svg的ps插件

将ps中的矢量文件导出为svg格式文件： https://github.com/janily/svgartisan

案例

.path{
    stroke-dashoffset: 660;
    stroke-dasharray: 660;
    animation:draw 4s linear infinite;
}
@keyframes draw{
    100% {
        stroke-dashoffset: 0;
    }
}
